# PAW_Recurso2021

Este projecto foi desenvolvido no âmbito da disciplina PAW (Programação em Ambiente Web)

Para dar start do projeto, correr os seguintes comandos dentro de "PAW_Recurso2021/Backend/" e "PAW_Recurso2021/Frontend/"
- npm i
- npm start

Aceder no browser em http://localhost:4200
Importar regions.json no MongoDB como "insert document".
Usei DB local, por isso, mudar ligação em app.js
